# Installation

```bash
$ git clone https://github.com/jinwatanabe/ushiro
$ cd ushiro
```

# Usage

```bash
# Run ushiro
$ make dev
```

# Test

```bash
# Run test
$ make test

# RUN lint check
$ make lint

# RUN auto formatter
$ make black
```
